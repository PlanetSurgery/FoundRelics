### Setup Instructions
   *Tip: This tool is still a WIP, so you may run into issues during setup and use, especially use. This is not fully tested. Some items do not work and run count bugs out after portal boss, so does not count boss runs at this moment; Run Time and the "Earned" labels are nulled as well.*

1. **Install Python:**  
   Ensure you have Python installed on your machine. If not, download it [here](https://www.python.org/downloads/).

2. **Initial Setup:**  
   Run the `Setup.bat` file to install any required modules.

3. **Settings:**  
   Make sure you have the "Settings>Options>Query API" turned on in game settings.

4. **Adding New Items:**  
   When providing new items, ensure that the file name exactly matches its name in the JSON data.
