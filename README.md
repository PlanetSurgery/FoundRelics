# FoundRelics
Lost Relics Loot Tracker Addon

Track your loot.
Know your earnings.

Want to elevate your UI while also keeping track of your earnings, stress free? Well have I got some news for you! FoundRelics is a loot tracker that utilizes Lost Relics' built in API, keeping track of all items you drop from adventures and containers without a worry.

This tool utilizes the built in API system provided by Lost Relics. You can get started by enabling the Query API option in the 2nd tab of the settings, as you can see below. You must be a member in order to use this feature; there is currently no way around this. Consider it a treat for supporting the game! 

This tool:
* Loot dropped from adventures.
* Loot dropped from containers.
* Provides 24h information on your latest earnings.
* Provides a collectible book to track all earned items, including DrLoot NFTs!
